This is Elective Tracking System Application

Project information

  • Category: Web design
  • Features:
      • Student Enrollment and Login
      Elective Options and Details Display
      Real-Time Seat Availability Updates
      Tracking and Confirmation of Elective Choices
      Admin Dashboard for Managing Electives
    Technologies Used:
      Frontend: HTML5, CSS3, JavaScript, and Bootstrap for creating a dynamic and user-friendly interface.
      Backend: Python (Flask/Django) or PHP for managing server-side operations and logic.
      Database: MySQL or PostgreSQL to store student data, elective details, and enrollment records.
      Authentication: OAuth or custom login system to ensure secure user access.
      Deployment: Deployed on platforms like Heroku, AWS, or a university server for accessibility.
